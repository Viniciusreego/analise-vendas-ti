# 📊 Análise de Vendas de Produtos de Informática

Projeto desenvolvido em Python para praticar análise de dados utilizando Pandas e Matplotlib.

## Objetivos

* Calcular faturamento total.
* Identificar produtos com maior faturamento.
* Comparar vendas entre cidades.
* Avaliar desempenho por categoria.

## Tecnologias

* Python
* Pandas
* Matplotlib

## Principais Análises

* Faturamento por produto.
* Faturamento por cidade.
* Faturamento por categoria.
* Visualização gráfica dos resultados.
